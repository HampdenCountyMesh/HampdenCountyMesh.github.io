WHY THIS CHANGED

“Infrastructure” was too broad as a top-level page. Some visitors want to know what is currently online. Some want to learn what a node or observer is. Some want to see coverage. Some want to know what is planned next.

Splitting that information makes the site easier to read and keeps the main navigation focused on the public project: what it is, how to learn, where coverage is being built, what is online, and what comes next.
