# Hampden County Mesh Discord

The Hampden County Mesh Discord is the main discussion and coordination space for MeshCore, Meshtastic, local radio, node help, field testing, infrastructure, and project work.

## Joining

Public join route:

https://hampdencountymesh.org/join/github-docs/

Use the website route instead of publishing the underlying Discord invite. This preserves source tracking and allows the invite to change without updating every public link.

For invite, onboarding, role, moderation, or access questions, contact:

[discord@hampdencountymesh.org](mailto:discord@hampdencountymesh.org)

## Getting Started

After joining:

1. Read `rules` and `server-guide`.
2. Complete onboarding when prompted.
3. Choose relevant interest and region roles.
4. Use `new-member-lobby` or `questions-help` when unsure where to begin.
5. Use `node-help`, `meshcore`, or `meshtastic` for device-specific questions.

Beginner questions are welcome. Prior radio experience and expensive equipment are not required.

When asking for technical help, useful information may include:

- Device model
- Firmware version
- Mesh system
- Current role or mode
- Relevant settings
- Phone app or computer software
- Exact error message
- What has already been tried
- General location when terrain or regional settings matter

Remove credentials, private messages, and unnecessary location details from screenshots and logs.

## Privacy, Security, and Moderation

Do not post:

- Credentials, private keys, tokens, or broker passwords
- Private messages
- Administrative addresses or access details
- Exact private node or installation locations
- Someone else’s personal information without permission

Sensitive security matters should be reported to:

[security@hampdencountymesh.org](mailto:security@hampdencountymesh.org)

Moderators may remove content, redirect discussions, restrict access, or remove members when necessary to enforce the server rules and Code of Conduct.

Private conduct or moderation reports may be sent to:

[discord@hampdencountymesh.org](mailto:discord@hampdencountymesh.org)

Do not post private reports in public GitHub issues or Discord channels.

See:

- `CODE_OF_CONDUCT.md`
- `SECURITY.md`
- `docs/data-access-and-observed-activity.md`
