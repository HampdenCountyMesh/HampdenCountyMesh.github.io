# Hampden County Mesh Discord

The Hampden County Mesh Network Discord is the main community space for mesh networking discussion, field testing, troubleshooting, documentation work, mapping, and project coordination.

The server is for people interested in MeshCore, Meshtastic, local radio, resilient communications, community infrastructure, and coverage testing across Hampden County and Western Massachusetts.

The Discord is used for:

## Privacy and location safety

Do not post exact private node locations without permission.

General areas are fine for coverage reports. Exact home addresses, private infrastructure locations, personally identifying location details, and sensitive deployment information should not be posted publicly.

Good examples:

* “Westfield area”
* “western Hampden County”
* “near Holyoke / Chicopee”
* “tested from a hilltop”
* “mobile test along a general route”

Avoid posting:

* home addresses
* exact GPS coordinates for private nodes
* private repeater locations
* private infrastructure locations
* photos that reveal addresses or sensitive locations
* someone else’s location without permission

When in doubt, generalize the location.

## Credentials and infrastructure safety

Do not post credentials, private keys, API tokens, webhook URLs, passwords, broker credentials, or anything that could compromise a device, account, server, node, broker, or service.

If you accidentally post sensitive information, delete it immediately and notify a moderator.

## Radio and legal expectations

Members are responsible for following applicable radio laws, service rules, and license requirements.

Do not encourage illegal transmitting, jamming, interference, impersonation, unauthorized access, or misuse of emergency, public safety, aviation, railroad, utility, commercial, or government systems.

The project is for lawful experimentation, education, local communications, and community infrastructure.

## Getting help

For general questions, use `#questions-help`.

For device problems, use `#node-help` and include as much useful detail as possible:

* device model
* firmware version
* role or mode
* relevant settings
* screenshots
* logs
* exact error messages
* what you already tried

If you are not sure where to ask, use `#new-member-lobby` or `#questions-help`.
