What is mesh?

Mesh radios pass short messages through nearby radios. A message can move from one node to another, helping people communicate across an area without relying on cell service or Wi-Fi.

Start here

Get a node, use the local settings, and start testing.

Coverage changes from place to place.

Hills, valleys, buildings, trees, and antenna height all affect what a node can hear. The map is useful, but real testing is what tells us what works.
