PROJECT STATUS

CORE INFRASTRUCTURE ONLINE

Hampden County Mesh currently operates documented project infrastructure including a MeshCore Room Observer, a Linux-based MeshCore-Hub, and local tools for logging, testing, and documenting network activity.

The wider county network is still being built. Additional observers, repeaters, coverage testing, community-hosted infrastructure, and public live data tools are in development.

LOCAL COMMUNICATIONS, LOCAL INFRASTRUCTURE
Mount Tom radio towers overlooking Hampden County.

Built around the places we actually live and operate.

Hampden County Mesh is focused on practical communications infrastructure for Western Massachusetts: local operators, local terrain, local testing, and locally maintained systems.

The goal is to build shared communications knowledge and useful local options for everyday experimentation, outages, emergencies, field work, and community coordination.

The project draws from mesh networking, radio experimentation, amateur radio, GMRS, CB, FRS, MURS, and emergency-minded community communications.
