# Asset Credits and Usage

This repository includes website source code, documentation, data files, and media assets for the Hampden County Mesh Network website.

## Photographs

Unless otherwise noted, photographs in `assets/images/` were taken by the Hampden County Mesh project maintainer and are used as project website assets.

Current project photographs may include:

* Mount Tom radio tower photographs
* Mount Tom State Reservation sign photographs
* Field testing photographs featuring MeshCore hardware
* Images of project equipment, radios, antennas, and node devices
* Other local communications, infrastructure, or community-related images

## Photo Reuse

Photographs and image assets in `assets/images/` are not automatically licensed for reuse outside the Hampden County Mesh Network website.

Please ask for permission before reusing, modifying, redistributing, republishing, training on, or incorporating project photographs or image assets into unrelated projects.

## Personal Images

Some project photographs may include the project maintainer, their hands, their personally owned equipment, or their personally owned devices.

These images are included for project documentation and website presentation. They should not be reused in unrelated projects, advertisements, training material, social media posts, derivative works, or public datasets without permission.

## Code and Documentation

Source code, documentation, and non-photo project files are covered by the repository license unless otherwise noted.

This asset notice is intended to clarify that the repository license does not automatically grant broad reuse rights for project photographs, personal images, or other media assets stored in `assets/images/`.

## Maps and Data

Project map data may include public boundary data, manually maintained project data, generalized locations, and future activity or coverage information.

Public map points may be generalized or omitted for privacy and safety. Published map data should not be treated as exact infrastructure location data unless explicitly stated.

## Third-Party Assets and Services

Third-party logos, maps, fonts, icons, libraries, external services, and referenced projects remain the property of their respective owners and are subject to their own licenses and terms.

This includes, but is not limited to:

* OpenStreetMap map tiles and attribution
* Leaflet
* Google Fonts
* GitHub Pages
* MeshCore
* Meshtastic
* External community links
* Any referenced radio, mesh, mapping, or device projects
