STEP 1 — WHAT IS A MESH NETWORK?

A mesh network is a group of radios that can pass messages between one another without requiring cellular service, internet access, or a central communications tower.

When you send a message from your device, it is transmitted over LoRa radio. If the destination is too far away to hear you directly, another node can receive the message and repeat it. Additional nodes can continue relaying the message until it reaches its destination.

For example:

Your Node → Repeater Node → Observer Node → Destination Node

Each participating node helps extend the reach of the network. As more nodes are deployed throughout an area, the network becomes larger, more resilient, and capable of reaching greater distances.

Unlike traditional repeater systems, mesh networks can use multiple paths through the network. If one node goes offline, messages may be able to find another route through neighboring nodes.

STEP 3 — INSTALL FIRMWARE

Most supported devices can run either MeshCore or Meshtastic.

While Hampden County Mesh focuses primarily on MeshCore, both platforms are widely used throughout the broader mesh networking community and are worth understanding.

For users interested in participating directly in Hampden County Mesh activities, MeshCore is currently the recommended starting point.

Follow the official installation instructions provided by your device manufacturer and the MeshCore or Meshtastic documentation.

STEP 4 — CONFIGURE YOUR NODE

After installing firmware, configure your node before attempting to join the network.

Important items to verify:

• Choose a unique node name
• Verify you are using United States frequency settings
• Verify your radio settings match the local network
• Test transmission and reception

Many connection issues are caused by devices using European or United Kingdom frequency plans instead of United States frequencies.

If you are unsure about your configuration, ask for assistance in Discord before making changes.

Don't worry about building the perfect setup immediately. Most operators continue refining their configuration as they gain experience.
