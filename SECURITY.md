# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability affecting the Hampden County Mesh Network (HCMN) website, GitHub repository, Discord server, infrastructure, services, or related tooling, please do not report it publicly.

Instead, contact:

[HampdenCountyMesh@protonmail.com](mailto:HampdenCountyMesh@protonmail.com)

Please include:

* Description of the issue
* Reproduction steps
* Potential impact
* Screenshots, logs, or other supporting information (if available)

## Scope

This policy applies to:

* HCMN websites
* HCMN GitHub repositories
* HCMN Discord infrastructure
* HCMN-hosted services and tools
* HCMN monitoring and observer systems

## Response

Reports will be reviewed as soon as possible. We will make a reasonable effort to acknowledge receipt of reports and investigate legitimate security concerns.

## Responsible Disclosure

Please allow a reasonable amount of time for investigation and remediation before publicly disclosing any security vulnerability.

Good-faith security research intended to improve the security of HCMN systems is appreciated.
