Mesh systems

Start with MeshCore. Meshtastic users are welcome.

Hampden County Mesh supports both MeshCore and Meshtastic. MeshCore is the main setup focus here because it fits the local direction for two-way messaging, fixed nodes, repeaters, and shared community use.

Meshtastic is still useful for learning LoRa basics, testing antennas and placement, telemetry, portable use, outdoor experiments, and casual participation.
