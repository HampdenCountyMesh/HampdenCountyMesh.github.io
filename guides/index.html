WHAT TO READ FIRST

If you are brand new, start with What Is Mesh? and Getting Started. Then read MeshCore Basics. If you already have a Meshtastic device, read Meshtastic Basics too.

If you are trying to understand local range, terrain, or signal checks, read Coverage and Signal Checks next. If you are coming from GMRS, amateur radio, CB, scanners, or another radio background, start with Radio Basics and the Radio Reference.

You do not need to know everything before joining the Discord. A lot of this is easier when you can compare notes with people nearby.
